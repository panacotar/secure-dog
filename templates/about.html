Secure Dog is an application that uses the Two Factor Authentication method to authenticate its users. Here, users can create an account and submit their favourite photos of animals with glasses.
Even though this is dog-themed, any photos of animals are welcomed as long as they wear glasses.

I chose this theme because I love dogs and I feel impartial about (sun)glasses, unless they're on animals, then I love (sun)glasses as well.

Secure Dog is my final project part of the CS50: Introduction to Computer Science course. Big thanks to all the fantastic teachers and staff putting together this course.

Enjoy!

Login instructions

  1. Register or log in with email and password
  2. Get your confirmation code by email
  3. Use the code to confirm your identity
  4. Get to the juicy part

Post a photo

  1. Search for an image
  2. Right-click on the image -> Press 'Copy Image Address'
  3. Copy the link in the "Photo URL" field

Technologies

Written in Python3 and built with Flask web framework, HTML templating was done using the Jinja template engine. As a first version, it utilizes SQLite3 as the Database. The app is currently hosted on Heroku.
